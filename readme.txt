== Description ==

Randomly Display Bangla meaning from Holy Al Quran

== Installation ==

1. Upload `BanglaAlQuran` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and Drop the Widget
4. or use <? do_shortcode(['bd-al-quran']); ?>
or put [bd-al-quran] as shortcode :)
